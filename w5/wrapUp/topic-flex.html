display: flex 에 수반되는 주요 css property

flex-direction
정렬 방향을 설정.
row : 가로 방향
column : 세로 방향
flex-wrap
자식 요소들이 정렬 방향을 따라 정렬되기에 넘칠 때 줄바꿈을 할지의 여부.
wrap : 줄 바꿈
nowrap : 줄 바꾸지 않음
justify-content
정렬 방향을 따라 자식들을 어떻게 배치할지에 대한 속성
start : 좌측 혹은 상단부터 배치
end : 우측 혹은 하단부터 배치
center : 중앙에 배치
space-around : 자식 요소를 양끝에 붙이고 등간으로 배치
space-between : 자식 요소의 좌우에 같은 간격을 주면서 배치
space-evenly : 자식 요소를 양끝에서 때고서 등간으로 배치
align-items
정렬 방향에 수직으로 자식들을 어떻게 배치할지에 대한 속성
start : 좌측 혹은 상단에 배치
end : 우측 혹은 하단에 배치
center : 중앙에 배치
stretch : 좌우를 가득 채우거나, 상하를 가득 채우도록 늘어남

부모요소의 크기, 정확히는 정렬 방향의 치수가 자식요소에 의해 결정되는지, 아니면 자식과 무관하게 결정되는지에 따라 적용할 수 있는 속성이 다르다.

부모의 크기는 자식과 무관 (가로)

child1
child2
child3

부모의 크기는 자식을 따름 (가로)

child1
child2
child3

부모의 크기는 자식과 무관 (세로)

child1
child2
child3

부모의 크기는 자식을 따름 (세로)

child1
child2
child3
